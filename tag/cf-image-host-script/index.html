使用独立图床子域名

最近在将 Wordpres 切换到 OctoPress，顺便将图片统一放到Upload4D管理。

我挑选图床管理程序有下面几个需求，需求由强到弱排列：

  • 开源
  • 简单
  • 不需数据库支持
  • 支持分目录
  • 允许上传图片
  • 支持后台直接操作文件
  • 支持用户管理，不允许其他人上传
  • 页面美观
  • 不要生成缩略图等文件
  • 软件持续更新
